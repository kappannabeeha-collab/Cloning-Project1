# boAt Lifestyle Website Clone

## Project Description
This project is an educational, full-stack clone of the **boAt Lifestyle** e-commerce website.  
The objective is to understand real-world web application architecture, frontendâ€“backend integration, and collaborative development using modern web technologies.

This clone is built **from scratch** for learning purposes only. No proprietary source code has been copied.

---

## Project Goals
- Recreate the core shopping experience of an e-commerce platform
- Practice team-based development using Git and GitHub
- Build a scalable frontend and backend architecture
- Implement authentication, product browsing, cart management, and checkout flow

---

## Scope

### Included Features
- Homepage
- Product listing page
- Product detail page
- User authentication (Register / Login)
- Cart functionality
- Checkout page (mock order success)

### Excluded Features
- Online payments
- Reviews and ratings
- Wishlist
- Admin dashboard
- Advanced animations

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Axios
- Context API

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- bcrypt

### Tools
- Git & GitHub
- Postman
- VS Code
